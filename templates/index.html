About the Project

How DermAI Works

A CNN-powered skin disease detection system built with TensorFlow/Keras and Flask, with a voice-enabled Claude AI chatbot for dermatology advice.

CNN Architecture
IN
Input
64 × 64 × 3 RGB — resized on server
↓
C1
Conv2D
32 filters · 3×3 kernel · stride (3,3) · ReLU activation
↓
P1
MaxPooling2D
Pool 2×2 → 11×11×32 = 3,872 features
↓
FL
Flatten
3,872 features → 1D vector
↓
D1
Dense — Hidden
500 units · ReLU activation
↓
OUT
Dense — Output
5 units · Softmax → class probabilities

Preprocessing (Critical Note)
# ⚠ NO /255 normalisation — matches original working prediction pipeline img = keras_image.load_img(filepath, target_size=(64, 64)) x = keras_image.img_to_array(img) # shape: (64, 64, 3) x = np.expand_dims(x, axis=0) # shape: (1, 64, 64, 3) preds = model.predict(x, verbose=0) # shape: (1, 5) label = INDEX[np.argmax(preds, axis=1)[0]]
